# Desktop App

Bootstrap Electron + React + TypeScript per il progetto desktop.

## Comandi utili

Da root monorepo:

- `./scripts/desktop-dev.sh` -> sviluppo renderer
- `FULL_ELECTRON=1 ./scripts/desktop-dev.sh` -> sviluppo Electron completo (serve GUI forwarding)
- `./scripts/desktop-build.sh` -> build produzione
